# Сайт для НКО "Старшие Братья, Старшие Сестры"

## Технологии используемые в проекте
* React JS

## Скрипты

`yarn build` Сборка проекта в директорию dist

`yarn start` Старт локального веб сервера [http://localhost:3000](http://localhost:3000)
